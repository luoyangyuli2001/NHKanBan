# NHKanBan

English | [中文](README.md)

A community task board mod for **GregTech: New Horizons** (Minecraft 1.7.10).

NHKanBan provides a Kanban-style task board where players can manually create,
claim, assign, and track tasks within a server.

This mod is designed as a pure collaboration tool.

> **Status: Early development.** Not yet usable. APIs and features are subject to change.

## Planned Features

- Kanban-style GUI with four columns: To Do / In Progress / Review / Done
- Task scopes: Global / Team / Direct
- Subtasks, priorities, deadlines, comments
- Task board block + portable task terminal item
- HUD reminders for claimed tasks and approaching deadlines
- Built on [ModularUI2](https://github.com/GTNewHorizons/ModularUI2)

## Requirements

- Minecraft 1.7.10
- Forge 10.13.4.1614
- ModularUI2
